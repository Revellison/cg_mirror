# ⛓️ cg Mirror Proxy

Высокопроизводительный микросервис на **Go** для организации зеркал загрузки файлов. Позволяет создавать красивые прямые ссылки (например, `client.example.ru/v7a`) для скачивания последних релизов с GitHub или любых других статических ресурсов.

---

## ✨ Особенности

*   **GitHub API Integration**: Автоматический поиск последнего релиза и выбор нужного файла по маске (архитектуре).
*   **Static Links Support**: Возможность прописать прямые ссылки на любые сторонние S3, Telegram или CDN хранилища.
*   **Multi-Port Architecture**: Запуск независимых инстансов прокси на разных портах для разных проектов в рамках одного бинарника.
*   **Caddy Ready**: Идеально работает в связке с Caddy для автоматического получения SSL (HTTPS).
*   **YAML Configuration**: Простая и понятная настройка всех репозиториев в одном файле.
