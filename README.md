# ⛓️ cg_Mirror Proxy (cg_mirror)

Легковесный и производительный микросервис на **Go**, предназначенный для создания «умных» зеркал загрузки. Он позволяет транслировать сложные или динамические ссылки (например, релизы GitHub) в простые и постоянные URL.

Например: `client.example.ru/v7a` —> автоматически перенаправляет на последний бинарник архитектуры v7a из вашего репозитория.

---

## ✨ Основные возможности

*   **GitHub API Integration**: Динамический поиск последних релизов. Вам не нужно обновлять ссылки вручную после каждого билда.
*   **Static Links Mapping**: Поддержка обычных прямых ссылок (S3, CDN, Telegram) для путей, не связанных с GitHub.
*   **Multi-App Support**: Возможность запускать несколько независимых сервисов на разных портах через один бинарник.
*   **Auto-SSL**: Полная совместимость с Caddy для автоматического получения сертификатов.
